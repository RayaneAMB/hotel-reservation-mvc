# 🏨 Hotel Reservation Management System

A robust desktop application built with **Java** and **Java Swing** for efficiently managing hotel bookings, guests, and invoicing. This project strictly follows the **MVC (Model-View-Controller)** architecture to ensure clean code separation, scalability, and easy maintainability.

## ✨ Features
* **Room Booking System:** Check room availability and create new reservations easily.
* **Guest Management:** Register and manage customer information.
* **Invoice Generation:** Automatically calculate and generate bills for guest stays.
* **Intuitive User Interface:** A clean, user-friendly graphical interface built entirely with Java Swing.

## 🛠️ Tech Stack
* **Language:** Java
* **GUI Framework:** Java Swing
* **Design Pattern:** MVC (Model-View-Controller)
